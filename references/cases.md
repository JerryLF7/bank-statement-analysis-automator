# Bank Statement Analysis Cases

This file stores confirmed special-case handling guidance for the bank-statement analysis workflow.

Use this file for case-specific patterns that are useful to remember but are too narrow or example-driven to belong in `reference.md`.

## What belongs here

- bank- or layout-specific handling
- page-span quirks
- statement-summary placement variations
- transaction-label patterns that apply only to a narrow case

## What does not belong here

- broad deposit or NSF rules
- durable month-assignment logic
- default worksheet mapping
- general completion requirements

Add new entries as concise, reusable case notes.
